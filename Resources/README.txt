## Fitur

- Fitur 1: Menjalankan pencarian
- Fitur 2: Pencarian berbasis AI

## Persyaratan Sistem

- OS: Windows, Linux, atau macOS
- Versi Python: 3.8 atau lebih tinggi (jika proyek ini menggunakan Python)
- Dependensi: Daftar pustaka atau modul yang diperlukan.

## Instalasi

1. Jalankan file `Firecat.exe` (untuk Linux/macOS) atau `Firecat.exe` (untuk Windows) untuk memulai instalasi.
2. Taruh Shortcut di Desktop.
